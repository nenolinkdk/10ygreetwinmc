# Birthday Block Mission til Windows

Greet and win med Minecraft-tema til en 10-års fødselsdag.

Et lille dansk matematikspil til Windows med Minecraft-inspirerede blokke.

## Installer

Download og kør:

`Install-BirthdayBlockMission.cmd`

Installationsfilen kopierer spillet til:

`%LOCALAPPDATA%\BirthdayBlockMission`

Den laver også genveje på skrivebordet og i Start-menuen.

## Spillet

- 10 matematikmissioner
- Plus, minus, gange og lidt division
- Tre svarblokke
- Rigtigt svar giver 10 point
- Scoren bygger små blokke fra sand til diamant
- Rigtigt svar viser Steve
- Forkert svar viser creeper, zombie eller skelet
- Slutside med fødselsdagskage og levende lys
- Ingen login
- Ingen server
